35 *The worker class for the conesearch module* 

36 

37 **Key Arguments:** 

38 - ``log`` -- logger 

39 - ``dbConn`` -- a database connection 

40 - ``tableName`` -- the name of the database table to perform the conesearch on. 

41 - ``columns`` -- the columns requested from the database table 

42 - ``ra`` -- the right ascension of the conesearch centre, can be single value or list of values 

43 - ``dec`` -- the declination of the conesearch centre, can be single value or list of values 

44 - ``radiusArcsec`` -- radius of the conesearch to be performed in arcsecs 

45 - ``sqlWhere`` -- clause to add after "where" in the initial sql query of the conesearch. Default *False* 

46 - ``raCol`` -- the database table ra column name. Default * raDeg* 

47 - ``decCol`` -- the database table dec column name. Default *decDeg* 

48 - ``separations`` -- include the separations in the final output. Default *False* 

49 - ``distinct`` -- request distinct columns from the database table (i.e. *select DISTINCT ...*). Default *False* 

50 - ``closest`` -- return the closest match only. Default *False* 

51 

52 **Usage:** 

53 

54 Say we have 5 locations we wish to search a database table called *transientBucket* to see if it contains sources at those locations. Add the coordinates to those locations to RA and DEC lists like so: 

55 

56 .. code-block:: python 

57 

58 raList = ["23:25:53.56", "02:10:08.16", 

59 "13:20:00.00", 1.47329, 35.34279] 

60 decList = ["+26:54:23.9", "-48:38:24.3", 

61 "+24:18:00.00", 8.43016, -42.34428] 

62 

63 Note coorinates can be in decimal degrees or sexegesimal format (or both). 

64 

65 To initialise a 10 arcsec conesearch to return the *transientBucketId* and *spectralType* values from any resulting match use the code: 

66 

67 .. code-block:: python 

68 

69 from HMpTy.mysql import conesearch 

70 cs = conesearch( 

71 log=log, 

72 dbConn=dbConn, 

73 tableName="transientBucket", 

74 columns="transientBucketId, spectralType", 

75 ra=raList, 

76 dec=decList, 

77 radiusArcsec=10, 

78 separations=False, 

79 distinct=False, 

80 sqlWhere=False 

81 ) 

82

374 self, 

375 sqlQuery): 

376 """* execute query and trim results* 

377 

378 **Key Arguments:** 

379 - ``sqlQuery`` -- the sql database query to grab low-resolution results. 

380 

381 **Return:** 

382 - ``databaseRows`` -- the database rows found on HTM trixles with requested IDs

390 sqlQuery=sqlQuery,

399 databaseRows = readquery( 

400 log=self.log, 

401 sqlQuery=sqlQuery, 

402 dbConn=self.dbConn, 

403 quiet=True 

404 ) 

405 else: 

406 print(sqlQuery)

409 if self.distinct and (self.columns != "*" and (self.raCol.lower() not in self.columns.lower() or self.decCol.lower() not in self.columns.lower())):

430 def _list_crossmatch( 

431 self, 

432 dbRows): 

433 """*to a finer grain crossmatch of the input coordinates and the database results.* 

434 

435 **Key Arguments:** 

436 - ``dbRows`` -- the rows return from the database on first crossmatch pass. 

437 

438 **Return:** 

439 - ``matchIndices1`` -- indices of the coordinate in the original ra and dec lists 

440 - ``matches`` -- the matched database rows 

441 """ 

442 self.log.debug('starting the ``_list_crossmatch`` method') 

443 

444 dbRas = [] 

445 dbRas[:] = [d[self.raCol] for d in dbRows] 

446 dbDecs = [] 

447 dbDecs[:] = [d[self.decCol] for d in dbRows] 

448 

449 # 12 SEEMS TO BE GIVING OPTIMAL SPEED FOR MATCHES (VERY ROUGH SPEED 

450 # TESTS) 

451 mesh = HTM( 

452 depth=12, 

453 log=self.log 

454 ) 

455 

456 if self.closest: 

457 maxmatch = 1 

458 else: 

459 maxmatch = 0 

460 matchIndices1, matchIndices2, seps = mesh.match( 

461 ra1=self.ra, 

462 dec1=self.dec, 

463 ra2=np.array(dbRas), 

464 dec2=np.array(dbDecs), 

465 radius=float(old_div(self.radius, (60. * 60.))), 

466 maxmatch=maxmatch # 1 = match closest 1, 0 = match all 

467 ) 

468 

469 matches = [] 

470 

471 for m1, m2, s in zip(matchIndices1, matchIndices2, seps): 

472 if self.separations: 

473 dbRows[m2]["cmSepArcsec"] = s * (60. * 60.) 

474 matches.append(dbRows[m2]) 

475 

476 self.log.debug('completed the ``_list_crossmatch`` method') 

477 return matchIndices1, matches
